Interface ResourceRule

ResourceRule is the list of actions the subject is allowed to perform on resources. The list ordering isn't significant, may contain duplicates, and possibly be incomplete.

apiGroups

apiGroups: string[]
  • Defined in types/output.ts:4203

APIGroups is the name of the APIGroup that contains the resources. If multiple API groups are specified, any action requested against one of the enumerated resources in any API group will be allowed. "*" means all.

resourceNames

resourceNames: string[]
  • Defined in types/output.ts:4209

ResourceNames is an optional white list of names that the rule applies to. An empty set means that everything is allowed. "*" means all.

resources

resources: string[]
  • Defined in types/output.ts:4217

Resources is a list of resources this rule applies to. "" means all in the specified apiGroups. "‍/foo" represents the subresource 'foo' for all resources in the specified apiGroups.

verbs

verbs: string[]
  • Defined in types/output.ts:4223

Verb is a list of kubernetes resource API verbs, like: get, list, watch, create, update, delete, proxy. "*" means all.
