StatefulSetStatus represents the current state of a StatefulSet.

Properties

collisionCount

collisionCount: number
  • Defined in types/output.ts:3798

collisionCount is the count of hash collisions for the StatefulSet. The StatefulSet controller uses this field as a collision avoidance mechanism when it needs to create the name for the newest ControllerRevision.

conditions

conditions: StatefulSetCondition[]
  • Defined in types/output.ts:3803

Represents the latest available observations of a statefulset's current state.

currentReplicas

currentReplicas: number
  • Defined in types/output.ts:3809

currentReplicas is the number of Pods created by the StatefulSet controller from the StatefulSet version indicated by currentRevision.

currentRevision

currentRevision: string
  • Defined in types/output.ts:3815

currentRevision, if not empty, indicates the version of the StatefulSet used to generate Pods in the sequence [0,currentReplicas).

observedGeneration

observedGeneration: number
  • Defined in types/output.ts:3822

observedGeneration is the most recent generation observed for this StatefulSet. It corresponds to the StatefulSet's generation, which is updated on mutation by the API Server.

readyReplicas

readyReplicas: number
  • Defined in types/output.ts:3828

readyReplicas is the number of Pods created by the StatefulSet controller that have a Ready Condition.

replicas

replicas: number
  • Defined in types/output.ts:3833

replicas is the number of Pods created by the StatefulSet controller.

updateRevision

updateRevision: string
  • Defined in types/output.ts:3839

updateRevision, if not empty, indicates the version of the StatefulSet used to generate Pods in the sequence [replicas-updatedReplicas,replicas)

updatedReplicas

updatedReplicas: number
  • Defined in types/output.ts:3845

updatedReplicas is the number of Pods created by the StatefulSet controller from the StatefulSet version indicated by updateRevision.
