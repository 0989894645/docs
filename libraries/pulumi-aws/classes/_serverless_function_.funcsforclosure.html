Class FuncsForClosure

FuncsForClosure collects all the function defintions needed to support serialization of a given Closure object. Context is the shape of the context object passed to a Function callback. Note that a Closure object can reference other Closure objects and can also have cycles, so we recursively walk the graph and cache serialized nodes along the way to avoid cycles.
